# Лабораторные работы по C++

**Направление:** 11.03.04 Электроника и наноэлектроника  
**Университет:** Санкт-Петербургский горный университет  
**Год:** 2020

## Кратко
6 лабораторных работ по C++ от основ до структур данных. Каждое задание — отдельный модуль. Переключение через условную компиляцию.

## Задания
1. **Основы** — типы данных, ввод-вывод, файлы  
2. **Управление** — ветвления, циклы, switch  
3. **Массивы** — 1D/2D, операции, транспонирование  
4. **Сортировки** — выбором, вставками, пузырьком  
5. **Функции** — рекурсия, Fibonacci, массивы  
6. **Структуры** — база данных студентов, поиск, сортировка

## Быстрый старт
1. Клонируйте репозиторий
2. Откройте в Visual Studio 2022
3. В `main.cpp` задайте нужное задание:
```cpp
constexpr TaskID CURRENT_TASK = TaskID::TASK_1;  // ← Выберите TASK_1-TASK_6
```
4. Скомпилируйте (F5)

## Исходные данные
Включены тестовые файлы:
- `input_task2.txt` — начальное значение
- `input_task3.txt` — массивы 4×3
- `input_task4.txt` — 25 чисел
- `input_arrays.txt` — массивы 2×5
- `students_database.txt` — 40 студентов
